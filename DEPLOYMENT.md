# Deployment Instructions - AI Writing Tools

## One-Click Deploy to Render

This Next.js SaaS app is configured for deployment to Render's free tier with PostgreSQL database.

### Prerequisites
- GitHub account connected to Render
- Anthropic API key (for Claude API)
- Stripe account (optional for payment features)

### Steps

1. **Go to Render Dashboard**
   - Visit [render.com](https://render.com) and sign in with GitHub

2. **Deploy from Blueprint**
   - Click "New +" → "Blueprint"
   - Select repository: `Kevo919/ai-writing-tools`
   - Branch: `main`
   - Click "Apply"

   Render will automatically create:
   - ✅ Web Service (Next.js app)
   - ✅ PostgreSQL Database (free tier, 1GB)
   - ✅ Database connection (auto-configured)

3. **Configure Environment Variables**

   **Required:**
   - `NEXTAUTH_URL` - Your deployment URL (e.g., https://ai-writing-tools.onrender.com)
   - `NEXTAUTH_SECRET` - Auto-generated by Render
   - `ANTHROPIC_API_KEY` - Your Anthropic API key from console.anthropic.com

   **Optional (for paid features):**
   - `STRIPE_SECRET_KEY` - Stripe secret key
   - `STRIPE_PUBLISHABLE_KEY` - Stripe publishable key
   - `STRIPE_WEBHOOK_SECRET` - Stripe webhook signing secret

   **Auto-configured:**
   - `DATABASE_URL` - PostgreSQL connection string (from database service)
   - `NODE_ENV` - Set to `production`

4. **Database Migration**
   - Render runs `npx prisma generate` during build
   - After first deployment, run migration manually:
     - Go to service → Shell
     - Run: `npx prisma migrate deploy`

5. **Wait for Deployment**
   - Initial build takes 5-7 minutes (Next.js compilation)
   - You'll get a URL like: `https://ai-writing-tools.onrender.com`

6. **Test the App**
   - Visit your deployed URL
   - Create an account
   - Test writing tools (requires ANTHROPIC_API_KEY)

### Running with Degraded Features

If you don't have API keys yet, the app will still deploy but:
- ❌ AI writing tools won't work (need ANTHROPIC_API_KEY)
- ❌ Payments won't work (need Stripe keys)
- ✅ Landing page and UI will work
- ✅ User signup/login will work (uses database)

### Free Tier Limitations

**Web Service:**
- Sleeps after 15 minutes of inactivity
- 750 hours/month runtime
- First request after sleep: 30-60 seconds

**Database:**
- 1GB storage (PostgreSQL)
- Shared CPU
- Good for ~10,000 users

### Alternative: Deploy to Railway

If you prefer Railway:

1. Visit [railway.app](https://railway.app) and connect GitHub
2. Create new project from `Kevo919/ai-writing-tools`
3. Add PostgreSQL database from Railway marketplace
4. Configure environment variables (same as above)
5. Run `npx prisma migrate deploy` after first deployment

### Getting API Keys

**Anthropic API Key:**
1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Sign up for an account
3. Get $5 free credits
4. Create API key in settings

**Stripe Keys (Optional):**
1. Go to [stripe.com](https://stripe.com)
2. Create account
3. Use test mode for development
4. Get publishable and secret keys from dashboard

### Post-Deployment

After deployment:
1. Update `NEXTAUTH_URL` with your actual Render URL
2. Redeploy service to apply changes
3. Test all features end-to-end
4. Update [ONL-63](/ONL/issues/ONL-63) with live URL and mark as done
